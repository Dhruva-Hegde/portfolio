# Dhruva Hegde — 3D Software Developer Portfolio

A high-performance, immersive 3D portfolio showcasing software development projects, built with a "Glassmorphism & Cyberpunk" aesthetic.

## 🚀 Features

- **Immersive 3D Background:** Interactive global environment with orbiting technology badges built using Three.js and `@react-three/fiber`.
- **Secured Contact Form:** Integrated with Web3Forms, featuring Honeypot bot protection and Click-to-Reveal privacy for personal data.
- **Responsive Design:** Fully fluid layout optimized for mobile, tablet, and desktop.
- **Dynamic Theming:** Seamless dark and light mode toggle with persistent user preferences.
- **Modern Tech Stack:** Built with React 18, TypeScript, Tailwind CSS, and Framer Motion.

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **3D Engine:** Three.js, React Three Fiber, Drei
- **Animations:** Framer Motion, Tailwind Animate
- **Styling:** Tailwind CSS (Custom Neon/Glassmorphism tokens)
- **Forms:** Web3Forms API
- **Deployment:** Vercel

## 📦 Getting Started

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/Dhruva-Hegde/portfolio.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Setup Environment Variables:**
    Create a `.env` file in the root and add your Web3Forms Access Key:
    ```env
    VITE_WEB3FORMS_KEY="your_key_here"
    ```
4.  **Run locally:**
    ```bash
    npm run dev
    ```
